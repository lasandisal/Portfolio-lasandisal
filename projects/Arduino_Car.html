Bluetooth-Controlled Obstacle Avoiding Robot

An autonomous rover developed for the 'Robotic Application Development' module, featuring real-time obstacle evasion and remote Bluetooth override.

Module Objective

The core objective was to synchronize an HC-SR04 Ultrasonic sensor with a servo-driven scanning routine. The logic, written in C++ via the Arduino IDE, determines the safest path forward while allowing for manual intervention through a Bluetooth Control** interface.

System Specs

  • Development Env: Arduino IDE
  • Logic Language: C++ (Arduino Sketch)
  • Remote Protocol: Bluetooth Serial (HC-05)
  • Sensor Array: Ultrasonic & Multi-LEDs
  • Driver Logic: Dual H-Bridge PWM control
